## To install current verion of all packages:

>**yarn install --frozen-lockfile**

## To launch project:

>**yarn start**

## To develop project:

>**yarn dev**


## To debug project:

>**yarn debug**


## To build project:

>**yarn build**

## To test project:

>**yarn test**

## To test project silently:

>**yarn tests**
